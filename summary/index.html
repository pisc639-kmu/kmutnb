Summary

  • Sets
  • Conditional Set $$\{x|x \in \mathbb{N} \land x<10\} = \{1,2,3,4,5,6,7,8,9\}$$
  • Union of Sets $$\{1,2,3\} \cup \{2,3,4\} = \{1,2,3,4\}$$
  • Intersection of Sets $$\{1,2,3\} \cap \{2,3,4\} = \{2,3\}$$
  • Set Complement $$\mathbb{U} = \{1,2,3,4,5,6\}$$ $$\{1,2,3\}' = \{4,5,6\}$$
  • Element of $$1 \in \{1,2,3\}$$ $$4 \notin \{1,2,3\}$$
  • Subset $$\{1,2,3\} \subset \{1,2,3,4\}$$ $$\{1,3,5\} \not\subset \{1,2,3,4\}$$
  • Super set $$\{1,2,3,4\} \supset \{1,2,3\}$$ $$\{1,2,3,4\} \not\supset \{1,3,5\}$$
  • Powerset $$P(\{1,2,3\}) = \{\emptyset, \{1\}, \{2\}, \{3\}, \{1,2\}, \{1,3\}, \{2,3\}, \{1,2,3\}\}$$
  • Real Numbers
  • Integer $$\mathbb{Z} = \{\cdots,-2,-1,0,1,2,\cdots\}$$
  • Positive Integer, Natural Numbers \mathbb{Z}^+, $$\mathbb{N} = \{1,2,3,4,5,\cdots\}$$
  • Negative Integer $$\mathbb{Z}^- = \{-1,-2,-3,\cdots\}$$
  • Zero Integer $$\{0\}$$
  • Rational Number $$\mathbb{Q} = \{\frac{a}{b} | a,b \in \mathbb{Z}, b \neq 0\}$$
  • Irrational Number $$\mathbb{I} = \mathbb{R} \setminus \mathbb{Q} = \{\sqrt{2},\pi,e,\cdots\}$$
